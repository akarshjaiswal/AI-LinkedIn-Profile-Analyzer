# 🧠 AI LinkedIn Profile Analyzer

Ek smart web application jo **Google Gemini AI** ka use karke aapki LinkedIn profile ko analyze karti hai aur recruiters ke hisaab se suggestions deti hai.

## 🚀 Live Demo
Aap is project ko yahan dekh sakte hain:  https://ai-linked-in-profile-analyzer.netlify.app/

## ✨ Features
* **AI Analysis**: Aapki profile ka score (percentage mein) batata hai.
* **Smart Suggestions**: Profile mein kya galti hai aur use kaise sudharein, AI batata hai.
* **Enhanced Content**: AI aapke liye ek behtar Headline aur About section generate karta hai.
* **PDF Report**: Analysis ke baad aap apni report PDF mein download kar sakte hain.
* **User Auth**: Signup aur Login functionality ke saath.

## 🛠️ Tech Stack
* **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
* **Backend**: Python (Flask)
* **AI Model**: Google Gemini API
* **Database**: MongoDB
* **Deployment**: Render (Backend) & GitHub/Netlify (Frontend)
